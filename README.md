# GNN SLR reproducibility package

This repository package accompanies the application-oriented GNN SLR covering studies published from 2019 through **February 2026**.

## Important scope note
For readability, the manuscript refers to the coverage window as **2019--2026**. However, all 2026 values reflect only the **first two months of 2026** and should be interpreted as **partial-year counts** rather than full-year totals.

## Contents
- `protocol/` final protocol summary
- `search/` exact query strings, screenshots, and identified-record counts
- `screening/` coded screening log, exclusion codebook, PRISMA/source counts
- `data/` extraction workbook, cleaned paper-level files, figure source data, and data dictionary
- `figures/` cleaned figures
